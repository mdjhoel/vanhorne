Local intel

Locals have their own favourite ways to use the trail. Here are a few suggestions.

Routes
  • 19km Wharf to Wharf run

    Kurt organizes this run for Paddlefest. Not for the faint of heart!

  • 10km tour de St Andrews

    This route circumnavigates the town. Perfect for those with lots of energy!

  • Ministers island perimeter path (7km+)

    This route circumnavigates the Ministers Island. Make sure you time the tide right!
